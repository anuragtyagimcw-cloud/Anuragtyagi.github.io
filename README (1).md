## 🚀 Deploy to GitHub Pages

### Step 1: Create Repository
1. Go to [github.com/new](https://github.com/new)
2. Name the repository: `iamanuragtyagi.github.io` (replace with your GitHub username)
3. Set it to **Public**
4. Click **Create repository**

### Step 2: Upload Files
1. Click **"uploading an existing file"** on the empty repo page
2. Drag and drop `index.html` (and your photo if you have one)
3. Commit the changes

### Step 3: Enable GitHub Pages
1. Go to **Settings** → **Pages** (left sidebar)
2. Under **Source**, select `main` branch and `/ (root)` folder
3. Click **Save**
4. Your site will be live at: `https://YOUR-USERNAME.github.io`

## 📝 How to Customize

### Add Your Photo
Replace the placeholder block in `index.html` (search for `placeholder-avatar`) with:
```html
<img src="photo.jpg" alt="Anurag Tyagi" style="width:100%;height:100%;object-fit:cover;">
```
Upload `photo.jpg` to the same repository folder.

### Update Content
All content is in `index.html` — just edit the text directly. The sections are clearly labeled with HTML comments like `<!-- HERO -->`, `<!-- ABOUT -->`, etc.

### Custom Domain (Optional)
1. Go to Settings → Pages → Custom domain
2. Enter your domain (e.g., `anuragtyagi.com`)
3. Add a `CNAME` file to the repo containing your domain name
4. Configure DNS with your domain registrar
